## 🛠️ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/7silva/portfolio.git
```

2. Install dependencies:
```bash
bun install
```

3. Start the development server:
```bash
bun dev
```

4. Visit [http://localhost:3000](http://localhost:3000)
